# YouTube Ad Block Ultimate Experience

![Extension Icon](premium.png)

A lightweight and effective browser extension designed to completely eliminate all types of advertisements on YouTube. Enjoy an uninterrupted, premium-like viewing experience without the distractions.

## ✨ Features

This extension is built to be simple yet powerful, focusing on providing a clean and ad-free YouTube interface.

-   **Blocks All Video Ads**: Removes pre-roll, mid-roll, and post-roll video ads.
-   **Removes Banner Ads**: Cleans up the homepage, search results, and sidebar from banner and display ads.
-   **Hides Promoted Content**: Removes sponsored videos and "promoted" content from your feed.
-   **Bypasses Anti-AdBlock Walls**: Effectively circumvents YouTube's prompts to disable your ad blocker.
-   **Premium Logo**: Replaces the standard YouTube logo with the "Premium" version for a consistent look and feel (only for logged-in users).
-   **Enable/Disable Toggle**: Easily turn the extension on or off with a single click in the popup menu.

## 🚀 Installation

Since this extension is not on the Chrome Web Store, you can install it manually in any Chromium-based browser (like Google Chrome, Brave, Edge, etc.) by following these steps:

1.  **Download the Repository**:
    -   Click the green `Code` button on this GitHub page.
    -   Select `Download ZIP`.
    -   Extract the ZIP file to a permanent location on your computer

2.  **Enable Developer Mode in Your Browser** (Chromium): 
    -   Open your browser and navigate to the extensions page. You can usually find this at `chrome://extensions` & `brave://extensions`.
    -   In the top-right corner, turn on the **"Developer mode"** toggle switch.

3.  **Load the Extension**:
    -   Click the **"Load unpacked"** button that appeared.
    -   In the file selection window, navigate to and select the folder where you extracted the repository (e.g., the `Ultimate` folder).
    -   Click "Select Folder".

The extension is now installed and active! You will see its icon in your browser's toolbar.

## 🔧 Usage

The extension works automatically in the background. To manage it:

-   Click the extension's icon in your browser toolbar to open the popup.
-   Use the toggle switch to enable or disable the ad-blocking functionality.
-   When you toggle the switch, all open YouTube tabs will automatically reload to apply the changes.
